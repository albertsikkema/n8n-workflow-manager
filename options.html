n8n Configuration

How to create an n8n API Key:
  1. Go to your n8n instance Settings → API
  2. Click "Create API Key"
  3. Give it a descriptive name (e.g., "Chrome Extension")
  4. Copy the key (it will only be shown once)
The base URL of your n8n instance (without trailing slash)
Your API key is stored securely and cleared when you close the browser

GitHub Backup (Optional)

How to create a GitHub Personal Access Token:
  1. Go to GitHub Settings → Personal access tokens
  2. Click "Generate new token (classic)"
  3. Select scope: repo (Full control of private repositories)
  4. Click "Generate token" and copy it
Your token is stored securely and cleared when you close the browser
Format: owner/repository (e.g., myteam/n8n-backups)
